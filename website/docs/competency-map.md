# Competency Map

This map uses broad competency language that overlaps with common social-emotional learning and advisory frameworks. It is not an official standards alignment. Programs can adapt it to local standards, grade bands, and reporting systems.

Core competency groups used here:

- **Self-awareness**
- **Self-management**
- **Social awareness**
- **Relationship skills**
- **Responsible decision-making**
- **Systems thinking**
- **Digital/social information responsibility**

## Using the Map

This table works well for:

- parent or caregiver planning
- school or advisory program summaries
- low-stakes reporting on what students practiced
- matching lessons to local SEL or life-skills language

If you need evidence guidance to go with the map, use the [Assessment and Reflection Guide](/docs/assessment-and-reflection-guide).
